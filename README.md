This is an MVP project that uses React

-*Edit Freewheel Config*

-*Edit A/B Map Config*

## Installation

After cloning the repository, install dependencies:
```sh
yarn install
```

Now you can run your local server:
```sh
yarn dev
```
Server is located at http://localhost:3000/
